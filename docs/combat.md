# Combat / Formula / Balance

> สมองด้านตัวเลข — ดึงมาจาก 00-spec.md ใช้เป็นที่ทำงานเรื่อง balance
> อัปเดตที่นี่เมื่อจูนตัวเลขตอน playtest

## Formula (ระบบบวก)
ดาเมจ = ผลรวมแต้ม + โบนัส poker + โบนัสสี + โบนัสอุปกรณ์/ลูกน้อง

### โบนัส poker (เฉพาะลูกในช่องที่ลง) — ✅ validated Phase 0 (2026-06-20)
- เลขโดด +0 / Pair +5 / Two Pair +10 / Three Kind +15 / Full House +30 / Four +40 / Five +60
- **Straight แยกระดับ:** เรียง 3 ลูก +10 / 4 ลูก +20 / 5 ลูก +30
  (เปลี่ยนจาก flat +25 เพราะเรียง 3 เกิดบ่อย ~29% เมื่อมี 5 ลูก → flat ทำให้ทุ่มตี straight เป็นสูตรตายตัว)

### โบนัสสี (สีเดียวกันในกอง) — ✅ validated Phase 0
- 2 ลูก +5 / 3 ลูก +15 / 4 ลูก +30 / 5 ลูก +50
  (ปรับขึ้นจาก 5/10/20/40 เพราะสีหายากเท่า n-of-a-kind ทุกชั้น แต่เดิมจ่ายน้อยกว่า + reward ไม่ถึง)

### มอนสเตอร์ / Enrage / ของ — ✅ validated Phase 0 (prototype)
- **HP** = (70 + 10×เลเวล) × tier — General ×1, Elite ×2, Boss ×4 (round 5)
- **Intent** = (5 + 0.9×เลเวล) × tier — General ×1, Elite ×1.3, Boss ×1.7
- **Enrage (กันยื้อ):** เริ่มหลัง grace = ⌈HP/30⌉ + 3 เทิร์น แล้ว intent +4/เทิร์น → สู้ปกติไม่เจอ, ยื้อตั้งรับยันเต็มตายเอง
- ฐาน prototype: ผู้เล่น HP 120, 5 ลูกตายตัว (เกมจริงผู้เล่นแข็งขึ้นด้วยอุปกรณ์)
- defense ใช้สูตรเดียวกับ attack / สกิล = รวมแต้ม → ฟื้น HP (ยัง placeholder)

### 🔁 Phase 1 — เปลี่ยนเป็น level-based (จูนด้วย simulator `prototype/sim_play.js`)
> เหตุผล: แบบ pwr-scaling (Phase 0) ทำให้ของ/สกิล/ลูกน้อง "ไม่จำเป็น" (ชนะ 100% ใน 3 เทิร์น) → สลับเป็นเลือดมอนตามเลเวลคงที่ ผู้เล่นต้องแข็งขึ้นเองถึงผ่าน
- **HP มอน** = General `(16 + 4×เลเวล)` · Boss `×2.0` (round 5) — ต้นเกม L1-4 ≈ 20-32 (เหมาะ 1 เต๋า)
- **Intent** = General `round(3 + 0.8×เลเวล)` · Boss `×1.7`
- **เลเวลอัพ:** XP = เลเวลมอนที่ฆ่า · ขึ้นเลเวลเมื่อ XP ≥ `4 + lv×3` → +5 HP, +1 โจมตี/เลเวล (แก้ปัญหา "ตีแล้วล่องลอย")
- **การ์ด:** ดรอปจากบอส (แน่นอน) + มอนทั่วไป (12%) · สวมได้ 2 ช่อง เพิ่ม atk/def/hp
- **ลูกน้อง passive:** fixed ต่อชนิด (เลือก active 1 ตัว) บางตัวให้ "เต๋าสีนั้นในช่องโจมตี +X" (สียังทำหน้าที่เดียว = ความแรง ✓ ข้อ4)
- **Enrage:** grace = ⌈HP/refDmg[เต๋า]⌉ + 4 แล้ว intent +3/เทิร์น (refDmg≈[_,10,13,17,23,30] ตามจำนวนเต๋า)
- **ผู้เล่น:** HP = 30 + ของ/ลูกน้อง · เริ่ม **1 เต๋า** → ปลดถึง 5 จากบอสประจำโซน (4 โซน)
- **สกิล (เปลี่ยนจาก Heal):** Novice = 💢 ทุ่มพลัง — ผลรวมเต๋าในช่องสกิล ÷ band(2×จำนวนเต๋า) = Tier 1/2/3 → ดาเมจ `[4,9,18]` รวมเข้าดาเมจก้อนเดียว (ข้อ5) · Heal ยกให้ Acolyte (Phase 2)
- **ของสวม:** stat atk/def/hp รวมกับ passive ลูกน้อง (เลือก active 1 ตัว) · def = ลดดาเมจที่โดน
- **ยา:** 15 เงิน/+25 HP (sink) · พักฟื้น 20 เงิน

### 📊 ผล simulation (3000 รอบ, บอตเล่นด้วยสูตรจริง) — 2026-06-20
- (v2 หลังเพิ่มเลเวลอัพ + ลดเลือดต้นเกม) เล่นจบ **100%** · ตายเฉลี่ย **~0.26 ครั้ง/รอบ** · ไม่ตายเลย ~83% · เลเวลจบ ~11
- บอสเป็น "กำแพง" จริง: บอสป่า Novice เปล่า ชนะ ~2% / ฟาร์มแล้ว ~100% → **ของ/เลเวลจำเป็น** ✓
- ⚠️ ถ้าแก้ BAL ในเกม ต้องแก้ใน sim_play.js ให้ตรงกัน แล้วรันยืนยัน

### 🔧 Balance v4 — ปรับยากขึ้น + ระบบใหม่ (2026-06-22, ยืนยันด้วย sim)
- **ปรับยาก (#1):** Elite intent ×1.3→**×1.45**, Boss ×1.55→**×1.7** · ฟื้นเลเวลอัพ 12→**10** · ฟื้นหลังสู้ 15%→**10%**
  - ผล sim 3000 รอบ: **จบ 100% · ตายเฉลี่ย ~2.0/รอบ · ไม่ตายเลย 45% · เลเวลจบ ~12** (เดิม ~0.26 = ง่ายไป)
- **Regen (#3):** เดิน 1 ช่อง ฟื้น `ceil(maxhp×0.04)` (เมือง/โลก/เขต) — ไม่ฟื้นตอนสู้ (กัน turtle)
- **หนี (#5):** ผลรวมแต้มเต๋า ≥ `round(3×เต๋า + (Lvมอน−Lvเรา)×(เต๋า×0.5))` → หนีได้ · เลเวลเท่า=ครึ่งแต้มสูงสุด · พลาด=โดนสวน
- **สกิล Novice 3 แบบ (#4) — "เชื่อถือได้ ไม่พึ่งคอมโบ"** (เลือกในหน้า 👤): 💢 ทุ่มพลัง = แต้มรวม+(เต๋า×4) · 🩸 ดูดเลือด = แต้มรวม+(เต๋า×2), ฟื้นครึ่ง · 🛡️ กำบัง = แต้มรวม, เกราะ=แต้ม+(เต๋า×2)
  - เหตุผล: เลิกใช้สี/เลขซ้ำในสกิล เพราะ (ก) ซ้ำกับช่อง ⚔️ ที่คิด poker/สีอยู่แล้ว (ข) Novice baseline ควรมีเครื่องมือกดแล้วได้ผลแน่นอน ไม่ลุ้นดวงซ้อนดวง · gimmick ลุ้น ๆ ไป Phase 2 (3 อาชีพ)
- **ระบบ SP (2026-06-22):** maxSP = 6 + ⌊(lv−1)/2⌋ · ใช้สกิลเปลือง smash/drain 3 · guard 2 · ฟื้น +1/เทิร์น + เดินบนแมพ +1/ก้าว · พักโรงแรม/เลเวลอัพ/ยา SP เติมเต็ม · ถ้า SP ไม่พอ เต๋าในช่องสกิลเด้งไปช่องโจมตีแทน (ไม่เสียเปล่า)
- **ยา 1 ชิ้น/เทิร์น:** มียา HP (+25) และยา SP (+8) ใช้ได้เทิร์นละ 1 ชิ้น (C.itemUsed)
- **ปลดทอยซ้ำ (reroll) ออก** จาก Speed → parking ไว้เป็น option อนาคต (อาจให้ดรอปจากบอส) · Speed ตอนนี้เหลือหน้าที่เดียว = initiative (ฆ่าได้ไม่โดนสวน)
- **ปุ่มหนีย่อเล็ก** (.fleemini มุมขวา) กันกดผิด
- **Effect (#6):** slash⚔️ / straight=pierce🏹 / crit·สกิลแรง=smash💥 (ภาพ)
- **Drop rate (#2):** สารานุกรมโชว์เรททันทีที่เคยเจอ · research กั้นแค่รายชื่อของแน่นอน
