# ROADMAP — เกมลูกเต๋า

> แบ่งเป็น phase ใหญ่ ๆ แต่ละ phase มี "ประตูผ่าน" (gate) ที่ต้องผ่านก่อนไป phase ถัดไป
> **กฎเหล็ก:** ห้ามข้าม gate เพื่อไปทำของที่สนุกกว่า — ถ้า gate ยังไม่ผ่าน แปลว่ายังไม่พร้อม

## Phase 0 — Proof of Fun — ✅ ผ่านแล้ว (2026-06-20)

**เป้าหมาย:** พิสูจน์ว่า combat แกนกลางสนุกจริง ก่อนลงทุนอย่างอื่น

- [x] prototype combat: ทอย 5 ลูก → allocate ลงช่อง → คำนวณ damage ตาม formula (`prototype/combat_prototype.html`)
- [x] กราฟิกขี้เหร่ที่สุด (กล่อง+อิโมจิ) — ไม่แตะ art จริง
- [x] เล่นเอง ~20 เทิร์น → ตอบ "อยากทอยอีก" (revenge) + วางเต๋าคิดจริงเมื่อเห็นกฎ
- [x] ปิดช่องโกง (ตั้งรับยันเต็ม) ด้วย enrage-grace

**🚪 GATE: ✅ ผ่าน** — core มีของ + decision จริง + ไม่มี exploit → เปิด Phase 1 ได้
> ของแถมที่เพิ่มระหว่างพิสูจน์ความสนุก (เกินสเปก Phase 0): มอน tier/เลเวล, juice (ghost HP/ดาเมจเด้ง/สั่น), ของดรอปจิ๋ว, ปุ่มกฎการคิดแต้ม

## Phase 1 — MVP (เกมเล่นจบได้ 1 รอบ) — 🟡 prototype เล่นจบได้แล้ว (รอ gate คนจริง)

**เป้าหมาย:** เกมที่เล่นได้ตั้งแต่ต้นจนจบด่านแรก มีความก้าวหน้า
> สถานะ: ทำครบใน `prototype/phase1_loop.html` · ยืนยันเล่นจบ 100% ด้วย `prototype/sim_play.js`

- [x] 1 อาชีพ (Novice/Swordman baseline) + สกิล 💢 ทุ่มพลัง (tier)
- [x] ระบบปลดล็อกลูกเต๋า 1→5 (จากบอส 4 โซน)
- [x] มอนสเตอร์ 8 ทั่วไป + 4 บอส + ระบบจับเป็นลูกน้อง (research 10 + bait)
- [x] ระบบลูกน้อง (เลือก active 1 ตัว, passive) + สารานุกรม
- [x] เศรษฐกิจ: ดรอปเงิน → ซื้อยา / พักฟื้น
- [x] ระบบของสวมใส่ 5 ช่อง + ดรอป + paperdoll
- [x] map 4 โซน (ป่า/ทุ่งหญ้า/ถ้ำ/ภูเขาไฟ) + habitat + บอสจบเกม (มังกร)
- [ ] กราฟิกวาดมือจริง (art-director) — ยังเป็น emoji placeholder
- [ ] balance pass จากคนเล่นจริง (sim ผ่านแล้ว แต่ feel ต้องคนยืนยัน)

**🚪 GATE:** มีคนอื่นที่ไม่ใช่คุณเล่นแล้วเล่นจบโดยไม่งง + บอกว่าสนุก ← **ยังไม่ผ่าน (ต้องให้ทีมเทสต์)**
